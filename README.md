C Projects Collection

Overview

This repository contains a collection of C programming projects developed for learning programming fundamentals, problem solving, and data management concepts.

Projects Included

1. Student Management System

A program that stores and displays student information such as roll number, name, and marks using structures.

Features:

- Add student records
- Display student details
- Uses structures

---

2. Banking System

A simple banking application that allows users to deposit money, withdraw money, and check account balance.

Features:

- Deposit money
- Withdraw money
- Balance inquiry
- Menu-driven interface

---

3. Library Management System

A basic library management program that displays available books and manages book records.

Features:

- Display available books
- Simple library record management
- Easy-to-use interface

---

4. Contact Management System

A contact book application that stores names and phone numbers using structures.

Features:

- Add contacts
- Display contacts
- Uses structures for data storage

---

5. Tic Tac Toe Game

A console-based implementation of the classic Tic Tac Toe game for two players.

Features:

- Two-player gameplay
- Simple board display
- Turn-based system

---

Technologies Used

- C Programming Language
- GCC Compiler
- Console-Based Applications
